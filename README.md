Simple Django App
=============================
One page app that it show two graph

QUICK START
-----------
1. Install PostrgreSQL 10+
2. Install Python3.7.4+
3. Install all requirements
4. Create db for project
5. Put your data in tables
6. Enter command: python manage.py runserver
